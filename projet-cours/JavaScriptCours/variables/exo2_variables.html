Défis Express – Les variables JS, level 2

🎮 Défi 1 : Bonjour toi !

Consigne

  1. Crée deux variables prenom et nom.
  2. Crée une troisième variable message avec le texte "Bonjour, je m'appelle [prénom] [nom]".
  3. Affiche cette variable dans la console.

Pour ce défi, tu vas utiliser let et alert() ou console.log() pour afficher le résultat.

🎮 Défi 2 : Phrase complète

Consigne

  1. Crée trois variables : sujet, verbe, complément.
  2. Crée une quatrième variable qui assemble les variables pour former le texte "[sujet] [verbe] [complément].
  3. Affiche cette variable dans la console.

Pour ce défi, tu vas utiliser let, l'opérateur + et alert() ou console.log() pour afficher le résultat.

🎮 Défi 3 : Calcul de l’âge

Consigne

  1. Crée une variable anneeNaissance avec ton année de naissance (exemple : 1990).
  2. Crée une variable anneeActuelle avec l’année en cours (exemple : 2025).
  3. Calcule ton âge en soustrayant l’année de naissance à l’année actuelle.
  4. Stocke le résultat dans une variable age.
  5. Affiche dans la console : "Tu as [age] ans"

Pour ce défi, tu vas utiliser let, l'opérateur - et alert() ou console.log() pour afficher le résultat.

🎮 Défi 4 : L'addition simple

Consigne

  1. Crée deux variables a et b contenant chacune un nombre (exemple : 12 et 8).
  2. Crée une troisième variable somme qui contient le résultat de a + b.
  3. Affiche dans la console le message : "La somme est : [valeur]"

Pour ce défi, tu vas utiliser let, l'opérateur + et alert() ou console.log() pour afficher le résultat.

🎮 Défi 5 : Peut-on tout modifier ?

Consigne

  1. Crée une variable prenom avec ton prénom. Affiche-la dans la console.
  2. Change sa valeur pour un autre prénom, puis affiche-la à nouveau.
  3. Crée une variable age avec une valeur numérique (ex. 17).
  4. Ajoute 1 à cette variable (pour simuler une année de plus) en utilisant += 1 ou age = age + 1, puis affiche la nouvelle valeur.
  5. Crée une constante PI avec la valeur 3.14.
  6. Essaie de modifier PI (par exemple, en lui attribuant une nouvelle valeur).
    • → Que se passe-t-il ? Observe la console.

Pour ce défi, tu vas utiliser let, const, l'opérateur += et alert() ou console.log() pour afficher le résultat.

🎮 Défi 1 : Bonjour toi !

let prenom = "Laurent";
let nom = "Touchet";

let message = "Bonjour, je m'appelle " + prenom + " " + nom;
console.log(message);

🎮 Défi 2 : Phrase complète

// Crée trois variables : sujet, verbe, complément
let sujet = "Le chat";
let verbe = "dort";
let complement = "sur le canapé.";

// Assemble-les pour créer une phrase
let phrase = sujet + " " + verbe + " " + complement;
console.log(phrase);

🎮 Défi 3 : Calcul de l’âge

let anneeNaissance = 1990;
let anneeActuelle = 2025;

let age = anneeActuelle - anneeNaissance;
console.log("Tu as " + age + " ans");

🎮 Défi 4 : L'addition simple

// Crée deux variables a et b
let a = 12;
let b = 8;

// Calcule la somme
let somme = a + b;
console.log("La somme est : " + somme);

🎮 Défi 5 : Peut-on tout modifier ?

let prenom = "Léo";
console.log(prenom);
prenom = "Emma";
console.log(prenom); // OK

let age = 17;
age += 1;
console.log(age); // Simule une année de plus

const PI = 3.14;
console.log(PI);
// PI = 3.14159; // Provoque une erreur → const ne peut pas être modifié
